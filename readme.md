# Dark Mode Lover

**Dark Mode Lover** is a carefully crafted dark theme for Visual Studio Code, combining the best elements of One Dark and Monokai color schemes. Designed for optimal readability and comfortable long coding sessions with a balanced color palette and comprehensive language support.

## Installation

1. Open the Extensions view in VSCode (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for `Dark Mode Lover`
3. Click **Install**
4. Select the theme from `Preferences > Color Theme` and choose **Dark Mode Lover**

Alternatively:

```bash
# Via command palette
Ctrl+Shift+P (or Cmd+Shift+P)
> "Preferences: Color Theme"
> Select "Dark Mode Lover"
```

## Theme Structure

### Semantic Token Colors
- **Variables** - Standard, readonly, and library variants
- **Properties** - Declarations, readonly, static, and library support
- **Parameters** - With italic styling for emphasis
- **Functions & Methods** - Declarations, async, and library variants
- **Types & Classes** - Interfaces, enums, structs, and type parameters
- **Keywords & Modifiers** - Storage types, control flow, and decorators
- **Special Modifiers** - Wildcards for common patterns (declaration, deprecated, readonly, etc.)

### Token Colors (11 Categories)
1. **Comments** - Line, block, and documentation comments
2. **Constants** - Numeric, language (true/false/null), character escapes
3. **Strings** - Regular strings, templates, URLs, and regex
4. **Entities** - Types, classes, functions, tags, and components
5. **Keywords** - Control flow, storage, operators, and declarations
6. **Variables** - Standard, properties, parameters, and enum members
7. **Markup** - Headings, bold, italic, links, and diff highlighting
8. **Punctuation** - Braces, commas, semicolons, and template expressions
9. **Decorators** - Annotations and meta-programming constructs
10. **Invalid** - Error highlighting and deprecated code
11. **Language-Specific** - CSS, C#, SQL, Python, Java, and preprocessor directives

### UI Colors
- Editor workspace with balanced background and foreground colors
- Activity bar with clear indication of active tabs
- Sidebar with proper hierarchy and visual feedback
- Terminal with complete ANSI color support
- Diff editor with addition/deletion highlighting
- Bracket highlighting with 4-color rotation
- Selection and search result highlighting

## Configuration

The theme is fully configured out of the box. For custom adjustments, modify the settings in your VSCode preferences:

```json
{
  "workbench.colorTheme": "Dark Mode Lover",
  "editor.fontFamily": "Your preferred monospace font"
}
```

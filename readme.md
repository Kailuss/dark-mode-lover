## â—¾ Get Started in Seconds

1. Open **Extensions** in VS Code (`Ctrl+Shift+X`)
2. Search for **"Dark Mode Lover"**
3. Click **Install**
4. Select the theme: `Preferences > Color Theme > Dark Mode Lover`

**That's it!** Your editor is now transformed.
